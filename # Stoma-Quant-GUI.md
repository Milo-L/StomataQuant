# Stoma-Quant-GUI

## 项目简介

Stoma-Quant-GUI是一个专为植物气孔（stomata）分析设计的图形用户界面工具。该软件集成了计算机视觉和图像处理技术，能够自动检测、分割和量化植物叶片上的气孔，提供全面的形态学测量和分析功能。本工具为植物科学研究提供高效直观的分析手段，特别适用于气孔密度、大小、分布等参数的精确测量。

## 主要功能

- **图像导入与管理**：支持多种格式图像导入，多标签页浏览
- **AI辅助检测**：集成YOLO模型进行气孔自动检测与分割
- **手动标注工具**：提供多种标注工具（多边形、矩形、旋转矩形、线条、点）
- **测量与分析**：
  - 形态特征提取（面积、周长、圆形度等）
  - 比例尺设置与实际尺寸测量
  - 结果数据统计与导出
- **批处理功能**：对多张图像自动执行检测、分析任务
- **热图生成**：基于形态特征创建直观的热力图
- **数据导入/导出**：支持多种格式的标注数据导入与导出

## 安装指南

### 系统要求

- Windows 10或更高版本（也支持macOS和Linux）
- Python 3.8或更高版本
- 建议：独立显卡以加速AI检测（非必须）

### 安装步骤

1. 克隆仓库或下载源代码
   ```
   git clone https://github.com/yourusername/Stoma-Quant-GUI.git
   ```

2. 安装依赖库
   ```
   pip install -r requirements.txt
   ```

3. 运行应用程序
   ```
   python Load_Stoma_Quant_GUI.py
   ```

## 使用指南

### 载入图像

1. 点击"文件" → "打开"或使用快捷键Ctrl+O
2. 选择要分析的植物叶片图像
3. 图像将在主视图中显示，可通过标签页管理多张图像

### AI检测

1. 点击"AI" → "加载模型"选择预训练的YOLO模型
2. 点击"AI" → "推理设置"配置检测参数
3. 点击"AI" → "运行YOLO"执行自动检测
4. 检测结果会直接显示在图像上

### 手动标注

应用提供多种标注工具，位于左侧工具栏：
- 多边形工具：精确勾勒不规则形状
- 矩形工具：快速标注矩形区域
- 旋转矩形：标注任意角度的矩形
- 线条和点工具：特定测量需求

### 特征提取与测量

1. 先设置比例尺："工具" → "设置比例尺"
2. 选择需要测量的形状
3. 点击"分析" → "特征提取"
4. 结果将显示在右侧"测量结果"面板

### 批处理

1. 打开多个图像标签页
2. 点击"批处理" → "批处理操作"
3. 选择要执行的操作（AI检测、特征提取等）
4. 设置处理参数并开始批处理

### 热图生成

1. 完成特征提取后
2. 点击"分析" → "生成热图"
3. 选择特征类型和颜色方案
4. 设置输出位置并生成热图
